게임공학과 2014182015 박두환 - 동물농장


개발한 게임의 간략한 설명을 하자면
 데이빗이라는 돼지가 다른동물들을 해방하여 주인을 몰아내는것이 게임의 목적입니다.
마우스를 클릭하여 주인아저씨를 피해 맵을 돌아다니며 5개의 집중 동물들이 있는 4개의 집을 찾고 해당 집과 여러번 충돌하여 동물을 해방합니다
해방될 경우 오른쪽 위에 해당 동물의 아이콘이 사라집니다 .
3마리 이상의 동물이 해방되면  데이브는 승리합니다
아저씨에게 잡힐 경우 날짜가 하루씩 지나며 3일이 지나면 데이브는 잡아 먹히게 됩니다.

조작은 휠로 줌인 줌아웃 , 오른쪽클릭으로 이동입니다.
